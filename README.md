# 42-2022-transcendence

*This project is about creating a website for the mightly Pong contest !*

## Technologies used

**Front-End**
	*Framework: NextJS
	*Pong: PixiJS

**Back-end**
	*Database: PostgreSQL
	*Framework: NestJS
	*Socket: Socket.IO

## Launch the project

```
docker-compose up --build
```
