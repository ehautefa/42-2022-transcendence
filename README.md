# 42-2022-transcendence

*This project is about creating a website for the mightly Pong contest !*

___Technologies used___

**Front-End**
	* Framework: NextJS
	* Pong: PixiJS

**Back-end**
	* Database: PostgreSQL
	* Framework: NestJS
	* Socket: Socket.IO

___Launch the project___

```
docker-compose up --build
```
